# Luxurious Swing (Simplified Version)

**Luxurious Swing** is a professional multi-feature TradingView indicator that combines swing structure, Fair Value Gap (FVG) detection, and previous period levels into one clean, simplified tool.

It was created to help traders visualize price structure and smart money concepts efficiently without visual clutter.

---

## Features

### Swing Structure
- Detects swing highs and lows dynamically
- Labels Higher High (HH), Lower High (LH), Higher Low (HL), and Lower Low (LL)
- Adjustable swing period and label limit
- Automatic cleanup for broken structures

### Fair Value Gaps (FVG)
- Bullish and Bearish FVG detection
- Volume-based confirmation filter
- Configurable transparency, median line, and ATR normalization
- FVGs are removed automatically when filled

### Previous Period Levels
- Previous day/week/month high and low
- Wick and body median zones
- Automatic update on each new period
- Configurable visualization for medians and wick areas

---

## Parameters

- **Swing Period** – defines structure sensitivity  
- **FVG ATR Multiplier** – controls FVG detection threshold  
- **Volume Multiplier** – adds confluence based on volume  
- **Previous Period Timeframe** – daily, weekly, or monthly
